# Loading

```javascript
import Loading from 'mfcl/Loading'
```

Loaders help the user feel like things are happening

## Properties

- Pass down any props to the Loading Div
  <Props of={Loading} />

## Basic usage

<Playground>
  <Loading />
</Playground>

## Color

- Pass in any color hex or RGB

<Playground>
  <Row>
    <Loading color="blue" />
    <Loading color="red" />
    <Loading color="green" />
  </Row>
</Playground>

## Sizes

- sm, md, lg, or any pixel value

<Playground>
  <Row>
    <Loading size="sm" />
    <Loading size="md" />
    <Loading size="lg" />
    <Loading size="50px" />
    <Loading size="10px" />
  </Row>
</Playground>

## Can be used with a button

<Playground>
  <Button loading size="lg">
    Submit
  </Button>
</Playground>
