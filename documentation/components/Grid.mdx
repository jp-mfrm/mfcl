# Grid

```javascript
import Grid from 'mfcl/Grid'
import GridRow from 'mfcl/GridRow'
import GridItem from 'mfcl/GridItem'
```

You must import the css file if you want to use the Grid

```css
/* in your Global scss file: */
@import 'mfcl/layout-styles.scss';
```

## Description

Grid component used to deliver consistent content layouts across different screen sizes
