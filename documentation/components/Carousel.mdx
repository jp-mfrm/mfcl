# Carousel

```javascript
import { Carousel } from 'mfcl'
```

## Description

The Carousel component allows you to a display a selection of rotating items.

## Properties

<Props of={Carousel} />

## Basic Usage

<Playground>
  <Carousel ariaLabel="one">
    <div style={{ backgroundColor: 'rgb(214,52,38)', color: '#fff' }}>Slide 1</div>
    <div style={{ backgroundColor: ' rgb(253,204,11)' }}>Slide 2</div>
  </Carousel>
</Playground>

## Responsive Breakpoints

The Responsive prop overrides the provided props of the carousel at different pixel breakpoints

<Playground>
  <Carousel
    ariaLabel="one"
    itemsToShow={2}
    layoutGap={5}
    responsive={[
      {
        breakpoint: 700,
        itemsToShow: 1,
        hideIndicators: true,
        hideControls: true,
        layoutGap: 0
      }
    ]}
  >
    <div style={{ backgroundColor: 'rgb(214,52,38)', color: '#fff' }}>Slide 1</div>
    <div style={{ backgroundColor: 'rgb(253,204,11)' }}>Slide 2</div>
    <div style={{ backgroundColor: 'rgb(50,48,49)', color: '#fff' }}>Slide 3</div>
    <div style={{ backgroundColor: 'lightblue' }}>Slide 4</div>
  </Carousel>
</Playground>

### Custom Children

<Playground>
  <Carousel
    itemsToShow={1}
    hideIndicators
    disableControls
    variableWidth
    ariaLabel="one"
    capturePropagation="disable"
  >
   {Chips.list.map(({ label, value }) => {
    return (
      <div key={label}>
        <Chip label={label} variant="default" onClick={() => console.log(value)} />
      </div>
    )
  })}
  </Carousel>

  <Carousel
    itemsToShow={1}
    hideIndicators
    disableControls
    variableWidth
    layoutGap={5}
    ariaLabel="one"
  >
   <div style={{ backgroundColor: 'rgb(214,52,38)', color: '#fff', width: '300px' }}>Slide 1</div>
   <div style={{ backgroundColor: 'rgb(253,204,11)', color: '#fff', width: '300px' }}>Slide 2</div>

## Control Button Alignment

Any non conflicting combination of 'top', 'middle', 'bottom', 'left', 'right', 'center'.  
hideControls prop removes the control buttons from view until hovered.
