# Accordion

```javascript
import Accordion from 'mfcl/Accordion'
```

## Description

Accordion Component can be used to display a list of collapsable content panels holding either text or other React Components.

## Properties

<Props of={Accordion} />

## Basic Usage

<Playground>
  <Accordion
    items={[
      {
        title: 'Title 1',
        id: 'title1',
        content: mattressIpsum(2)
      },
      { title: 'Title 2', id: 'title2', content: 'Less interesting content' }
    ]}
  />
</Playground>

## With Preview Text

<Playground>
  <Accordion
    items={[
      {
        preview: 'title 1 Description',
        title: 'Title 1',
        id: 'title1',
        content: mattressIpsum(2)
      },
      { preview: 'more description', title: 'Title 2', id: 'title2', content: 'Less interesting content' }
    ]}
  />
</Playground>

<Playground>
